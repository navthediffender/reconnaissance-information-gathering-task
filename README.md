# Reconnaissance and Information Gathering Task

This project documents a cybersecurity reconnaissance and information-gathering task completed as part of my Level 3 Cyber Security Bootcamp training.

The task demonstrates how publicly available information and common security tools can be used to gather basic technical information about domains, DNS records, subdomains, web technologies, and indexed public documents.

## Tools Used

- Kali Linux
- Linux command line
- Ping
- NSLookup
- Assetfinder
- Wappalyzer
- Google search operators

## What I Did

- Checked whether a public domain was reachable using `ping`
- Used `nslookup` to identify a domain’s public IP address
- Researched recommended actions after an email appears in a data breach
- Used Assetfinder to identify publicly listed subdomains
- Used Wappalyzer to identify web technologies and JavaScript libraries
- Used Google search syntax to find publicly indexed PDF information

## Evidence

This repository includes:

- A written PDF report
- Redacted screenshots showing the commands, tools, and results

## Important Note

This was a training task using publicly available information only. No unauthorised access, exploitation, or attack activity was performed.
